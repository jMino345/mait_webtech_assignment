# College Knowledge Quiz (Bootstrap Version)

## Project Overview

The College Knowledge Quiz is an interactive web-based application designed to test users on their knowledge of college history and courses. The project is built using HTML5, Bootstrap 5, CSS animations, and JavaScript for logic and interactivity.

The quiz provides instant feedback, tracks scores, includes a timer, and offers a responsive and accessible user interface.

---

## Features

* Minimum 10 multiple-choice questions
* Four options per question (one correct answer)
* Instant feedback after each submission
* Score tracking system
* Animated progress bar
* Question timer (15 seconds per question)
* Question randomization
* Responsive design using Bootstrap 5
* Keyboard accessibility support
* Admin login section (basic password protection)

---

## Technologies Used

* HTML5 (Semantic Structure)
* Bootstrap 5 (Responsive Design and UI Components)
* CSS3 (Custom animations and transitions)
* JavaScript (Quiz logic and interactivity)

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` to open it in a web browser.

No additional installation is required.
